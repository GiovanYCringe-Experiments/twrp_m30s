# TeamWin Recovery Project for the Samsung Galaxy A50

Based on [Velosh/android_device_samsung_a50dd](https://github.com/Velosh/android_device_samsung_a50dd)

## How-to compile it:

To build:

```sh
. build/envsetup.sh
lunch omni_a50dd-eng
make recoveryimage
```
